 En los comentarios hay algunas letras entre paréntesis, esto son los botones
que se tiene que presionar.

(*) El poder es el mínimo para hacer el movimiento, luego de esto, el poder ira
disminuyendo, si queda muy poco poder el personaje quedara cansado.

 El tiempo de cansancio aumenta cuando el char esta haciendo el movimiento. El
personaje seguirá cansado hasta que se cumpla hasta que se cumpla el tiempo
establecido, esto no será cancelado por golpes del rival. Disminuye este tiempo
presionando cualquier botón.

 En una versión futura, si es posible, hacer que el uso de poder sea en
Porcentaje o partes del poder máximo, por ejemplo el Kamehame-Ha usara la 6°
parte del poder máximo, equivale a 500 si el poder máximo es 3000.

 Recomendaciones:
--------------------------------------------------------------------------------

 En el archivo "EGC-kid_Goku_GT.cns" se puede cambiar el número de estas lineas:

 +---------------------------------------+
 | Life = 3000                           |
 +---------------------------------------+
 | Vida maxima de este personaje.        |
 | El normal es 1000                     |
 +---------------------------------------+
 | Attack = 125	                         |
 +---------------------------------------+
 | Fuerza de ataque de este personaje.   |
 | El normal es 100                      |
 +---------------------------------------+
 | Defense = 125                         |
 +---------------------------------------+
 | Defensa de este personaje.            |
 | El normal es 100                      |
 +---------------------------------------+
 | Power = 3000                          |
 +---------------------------------------+
 | Poder maximo de este personaje.       |
 | El normal es 3000                     |
 | No es recomendable cambiar este valor.|
 | Si pones un número mas bajo puede que |
 | no se pueda ejecutar algunos ataques. |
 | Estoy trabajándo para que se pueda    |
 | cambiar al gusto que quieran.         |
 +---------------------------------------+

 La caracterisa pricipalmente de este personaje es de hacer combos, asi que 
presioana los botones de ataques para interrumpir un ataque y pasar a otro.
